# 📊 Sales Data Analysis Project

## 🚀 Overview

This project focuses on analyzing sales data using Python. It simulates a real-world business scenario where a data analyst extracts meaningful insights from raw data.

---

## 🎯 Objectives

* Calculate total revenue
* Identify top-selling products
* Analyze category-wise performance
* Evaluate city-wise sales
* Detect low-performing products

---

## 🧠 Key Features

* 📌 Data manipulation using Pandas
* 📌 Revenue calculation (`price × quantity`)
* 📌 Grouping using `groupby()`
* 📌 Filtering low sales data
* 📌 Sorting for better insights

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* Matplotlib (for visualization)
